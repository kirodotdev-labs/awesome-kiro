# Awesome Kiro [![Awesome](https://awesome.re/badge.svg)](https://awesome.re)

> A curated list of awesome projects, tools, and resources for [Kiro](https://kiro.dev) — the agentic IDE that works alongside you from prototype to production.

[Kiro](https://kiro.dev) is an agentic IDE built on VS Code that uses specs, agents, and hooks to help you go from idea to production. Learn more at [kiro.dev](https://kiro.dev) or check out the [documentation](https://kiro.dev/docs/).

## Official

- [Kiro](https://github.com/kirodotdev/Kiro) - The Kiro IDE — agentic development from prototype to production. Amazon-vended | Proprietary as of 2026-04-20
- [Kiro CLI](https://github.com/kiro-team/kiro-cli) - The command-line interface and agent engine powering Kiro. Amazon-vended | Apache-2.0 as of 2026-04-20
- [Powers](https://github.com/kirodotdev/powers) - Official Powers that add specialized context and tools to Kiro agents on-demand. Amazon-vended | Custom License as of 2026-04-20
- [Spirit of Kiro](https://github.com/kirodotdev/spirit-of-kiro) - A game built with and powered by generative AI — play it or help build it. Amazon-vended | MIT-0 as of 2026-04-20
- [AI-DLC Workflows](https://github.com/awslabs/aidlc-workflows) - Reference workflows for the AI Development Lifecycle. Amazon-vended | MIT-0 as of 2026-04-20

## Agent Configuration

### General-Purpose

- [jasonkneen/kiro](https://github.com/jasonkneen/kiro) - Comprehensive collection of Kiro skills, steering rules, and agent configurations. MIT as of 2026-04-20
- [kiro-generator](https://github.com/kiro-generator/kiro-generator) - Generate Kiro project scaffolding with specs, agents, and hooks. MIT as of 2026-04-20
- [kiro-project-template](https://github.com/BinarySword/kiro-project-template) - Starter template for Kiro projects with pre-configured agents and steering. MIT as of 2026-04-20
- [kiro-best-practices](https://github.com/awsdataarchitect/kiro-best-practices) - Curated best practices and patterns for working with Kiro agents. MIT as of 2026-04-20
- [claude-code-settings](https://github.com/feiskyer/claude-code-settings) - Agent configuration patterns portable across Kiro and other agentic tools. MIT as of 2026-04-20

### Domain-Specific

- [oh-my-kiro](https://github.com/1nhann/oh-my-kiro) - Opinionated Kiro configuration with security-focused steering and hooks. MIT as of 2026-04-20
